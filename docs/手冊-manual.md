## 一、這是什麼遊戲

《猴島小英雄 II：里察克的復仇》是 LucasArts（原名 Lucasfilm Games）於 1991 年推出的圖像冒險遊戲（graphic adventure），承接 1990 年首作《猴島小英雄》（The Secret of Monkey Island）的故事，主角依然是自稱「海盜」的笨拙青年蓋布拉許・特普伍德（Guybrush Threewood，官方原文拼法為 Threepwood，台版手冊沿用「特普伍德」音譯）。玩家操作蓋布拉許在加勒比海一帶的虛構島嶼間遊走，尋找傳說中的寶藏「大驚奇」（Big Whoop），同時躲避死而復生、對他恨之入骨的鬼船長里察克（LeChuck）。

遊戲採用 LucasArts 自家開發的 SCUMM（Script Creation Utility for Maniac Mansion）腳本引擎第五代版本，操作介面延續系列一貫的「動詞指令 + 物品」點選式設計，但相較初代做了精簡：動詞指令從 12 個減至 9 個，物品欄從純文字列表改為完全圖像化的圖示，並支援攜帶動物（狗、猴子）等新玩法。純 VGA 256 色畫面、iMUSE 動態配樂系統，是系列在技術與敘事上都更成熟的一代。

### 台灣代理與中文資料背景

本作在台灣由「軟體世界」代理發行，正式授權涵蓋中華民國與香港地區，包裝上標示「冒險珍藏版 156」、定價 NT$460，硬體需求為 IBM PC AT（或相容機）、640K RAM、VGA 256 色顯示、10MHz 以上速度、至少一台 1.2M 磁碟機及約 10MB 硬碟空間。台版包裝隨附贈品：一把印有「猴島英雄」字樣的鑰匙圈萬用刀（軟體世界當年的常見促銷贈品，包裝上寫著「馬蓋仙萬用刀一把，攻略本即將發行，送完為止」）。

當年軟體世界並未在包裝內附上完整劇情/物品攻略，僅在操作手冊末段留了一段「十二、誰來冒險？」的簡短提示，鼓勵玩家自行摸索解謎樂趣；正式攻略本則需另外向經銷商洽購。

隨著軟體世界代理權結束、原版磁片與紙本手冊逐年佚失或被丟棄，一群玩家（「骨灰集散地」，boneash.oldgame.tw）在民國 90 年代後自發性地發起掃描保存計畫，將軟體世界貴族版／珍藏版系列的說明書、封面、磁片等實體物件掃描建檔，避免這批早期中文電腦遊戲文化史料徹底消失。掃描檔案附帶的原始聲明文字（見 `軟體世界說明書補完計劃.txt`，Big5 編碼）大意為：

> 軟體世界是早期電腦遊戲發行商的佼佼者，它首創的精美 CD 盒包裝，讓電腦遊戲從不起眼的賠錢貨搖身變為商場上的金雞母。雖然這些遊戲早已不再由軟體世界代理，但早期帶給玩家的感動卻是難以抹滅的。隨著時間過去，這些早期的磁片和說明書也逐漸被丟棄，若再不發起一個有組織的行動，很多珍貴的說明書及資料將永遠消失。因此骨灰集散地發動了一個補完計畫……我們不會從這個活動中得到任何利益，甚至必須犧牲自己閒暇時間來掃描和整理。

本文所用的 30 張掃描圖，正是這個民間保存計畫的成果之一。我們在此基礎上整理成結構化中文文件，作為 ScummVM 中文化專案的歷史考據資料。

## 二、遊戲操作要點索引（SCUMM 介面）

MI2 的操作核心是「動詞（verb）+ 名詞（物品／人物）」的句子組合，畫面下方常駐一組指令表，玩家用滑鼠／鍵盤／搖桿把游標移到動畫視窗或物品欄中的物品上，選一個動詞，遊戲就會自動組出一句英文指令（例如 `Use stick with box`）並執行。

### 畫面組成

| 區塊 | 說明 |
|---|---|
| 動畫視窗 | 佔螢幕絕大部分，遊戲畫面、動畫、對話都在此呈現 |
| 文句列 | 動畫視窗下方，即時顯示目前組合出的指令句子 |
| 指令表 | 螢幕左下角，9 個動詞指令 |
| 物品欄 | 指令表右方，全圖像化顯示蓋布拉許攜帶的物品，超過 8 樣會出現捲動箭頭 |

### 9 個動詞指令（較初代的 12 個精簡）

| 代表鍵 | 英文 | 中文 |
|---|---|---|
| G | Give | 給予 |
| P | Pick up | 拿起 |
| U | Use | 使用 |
| O | Open | 打開 |
| L | Look at | 看 |
| S | Push (Shove) | 推 |
| C | Close | 關上 |
| T | Talk to | 交談 |
| Y | Pull (Yank) | 拉 |

值得注意：初代的「Walk to（走動）」指令在二代被拿掉了——因為太常用，直接把游標移到想去的方向、按動作鍵即可行走，不需要額外選指令。

### 三種操作方式

- **鍵盤**：方向鍵或數字鍵（小鍵盤 8 方向）移動；`Enter` 為選擇鍵，`Tab` 為動作鍵，`Esc` 跳過過場動畫。
- **滑鼠**：移動游標即為移動方向；左鍵＝選擇鍵，右鍵＝動作鍵。
- **搖桿**：類比方向對應移動；發射鈕 1＝選擇鍵，發射鈕 2＝動作鍵。
- 三種輸入方式可並用切換：`Ctrl+M` 切換滑鼠控制，`Ctrl+J` 切換搖桿控制（須先校正）。

### 其他常用控制鍵

| 按鍵 | 功能 |
|---|---|
| F5 | 存檔／讀檔選單（Save / Load / Play / Quit） |
| F8 | 重新開始遊戲 |
| 空白鍵 | 暫停／繼續 |
| `[` / `]` | 音量調小／調大（依音源裝置不同，或開關 PC 喇叭音效） |
| `+` / `-` | 加快／減慢訊息顯示速度 |
| `.`（句點） | 清除文句列 |
| Ctrl+X 或 Ctrl+C | 跳出遊戲 |
| Ctrl+V | 查看遊戲版本 |

### 存檔／讀檔

按 `F5` 叫出選單，四個選項：`Save`（儲存）、`Load`（讀取）、`Play`（回到遊戲）、`Quit`（跳出遊戲）。存檔採用槽位（slot）制，可用方向鍵在槽位間選擇，右鍵可在第一與最後槽位間快速切換；讀取進度會覆蓋掉目前正在玩的進度，官方手冊特別提醒「想再回到現在，別忘了先儲存」。

### 遊戲設計哲學（手冊原文摘要，第 27 頁）

台版手冊特別用一整節介紹製作團隊的設計理念：遊戲不會讓玩家無故莫名死亡，物品也不會因為「太尖銳」而傷到玩家；探索與發現才是解謎的樂趣所在，不鼓勵靠「反覆死亡重來」來硬解謎題；對話選項答錯也不會受到嚴厲懲罰。這段文字反映了 Ron Gilbert 團隊在 90 年代初期對「死亡冒險遊戲」（如同期 Sierra 遊戲常見的動輒卡死／死亡）的反思，是 LucasArts 冒險遊戲一貫「不整死玩家」的設計主張。

## 三、劇情背景（不劇透結局）

故事發生在蓋布拉許擊敗鬼船長里察克、抱得美人歸（市長艾蓐妮・瑪蕾，Elaine Marley）多年之後。蓋布拉許當年的「英勇」事蹟早已被以訛傳訛，扭曲得面目全非，沒人再相信他真的打敗過一個鬼船長；更糟的是，坊間開始流傳里察克其實已經復活的傳聞。

不甘寂寞的蓋布拉許決定重出江湖，探明真相，順便給自己找點新鮮刺激的冒險題材（「唬人」之用）。他踏上了臥虎藏龍的「無賴島」（Scabb Island），為了自保，必須設法找到加勒比海最龐大、最神秘的寶藏「大驚奇」（Big Whoop）——傳說這筆寶藏不只是財富，還藏著另一個世界的秘密，找到它，就能讓蓋布拉許永遠擺脫陰魂不散的里察克。

遊戲一開始，玩家會看到蓋布拉許被吊在半空中、進退不得，向舊情人艾蓐妮訴說自己怎麼落到這步田地——整段故事其實是倒敘：真正的冒險，要從無賴島的木蝨市集（Woodtick）開始說起。首要任務是離開無賴島，在 Phatt、Booty、Scabb 三座島嶼間遊走，尋找失散的四張藏寶圖。

## 四、Mix 'N' Mojo 材料比例轉盤（俗稱防拷密碼盤）

### 名稱更正說明

MI2 隨附的紙質轉盤，中文玩家社群常直接沿用「Dial-A-Pirate」這個名稱，但那其實是**前作**《猴島小英雄》（初代）的防拷轉盤名稱。二代所附的實際上是另一個機制：**"Mix 'N' Mojo" Voodoo Ingredient Proportion Dial**（巫毒材料比例轉盤），台版手冊統一稱之為「巫毒教獨家秘方轉盤」。掃描圖 `密碼盤001.jpg` 上印的標題也清楚寫著 `MIX 'N' MOJO` 與 `VOODOO INGREDIENT PROPORTION DIAL`，並非 Dial-A-Pirate。本文件依掃描原件更正這個常見誤稱。

### 機制是什麼、為何存在

這是 1990 年代初期軟體業界常見的**防拷保護（copy protection）**手段之一：遊戲不靠序號或磁片加密，而是要求玩家對照一份「只有正版包裝內才有」的實體紙製轉盤才能繼續遊戲，藉此讓非法拷貝磁片的玩家卡在起點過不了關（手冊裡的猴子對話「你過不了關而鬱卒嗎？」正是在調侃這件事）。轉盤是雙層可旋轉的圓盤，外環印著一圈「巫毒配料」圖示（蟲、疣、疤痕、蠟等等），內環則是密碼數字。

### 遊戲開頭如何使用

1. 啟動遊戲後，畫面上出現兩個不停旋轉的「怪東西」，按 `Enter` 鍵或滑鼠左鍵讓它們停下——這就是要求輸入密碼的畫面。
2. 螢幕左邊會顯示一個物品圖案，玩家要在轉盤**下層**的外緣找到對應圖案。
3. 螢幕右邊顯示另一個物品圖案，在轉盤**上層**的外緣找到對應圖案。
4. 旋轉內層圓盤，把這兩個圖案對齊、不要再移動。
5. 此時轉盤中央附近會出現一個「配料秘方」名稱（對應畫面上 `RECIPE FOR ___:` 的空格文字）。
6. 在該秘方名稱正上方的框格內，會看到兩組數字，那就是密碼。
7. 依序輸入這兩組數字，正確即自動進入遊戲；輸入錯誤還有幾次重試機會。

手冊附了一個實例：配料秘方為 `GOUT`（痛風）時，密碼是 `46/22`。

### 中文化版本的處理

ScummVM 引擎在重製／中文化版本中通常會直接略過這類防拷機制（因為原始磁片保護本來就是針對盜版磁片，正版數位化重玩不需要它），但基於「保全歷史」的原則，本手冊仍完整保留其原始說明與掃描圖，作為 1990 年代軟體防拷手段的歷史紀錄。
