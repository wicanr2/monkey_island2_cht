# 猴島小英雄 II 翻譯指令（所有翻譯 subagent 開場先讀這份）

你在翻譯《Monkey Island 2: LeChuck's Revenge》(1991, LucasArts SCUMM v5) 的遊戲文字成**繁體中文**。

## 風格尺度（已與專案負責人校準）

1. **適度台式瘋趣**：忠於原意但自然口語化。MI2 對白充滿雙關、諷刺、海盜吐槽——把英文的機智轉成**中文讀起來也好笑**的說法，雙關/梗適度在地化（可用台式口語），但**不硬塞梗、不過度鄉民化**，也別為了搞笑扭曲劇情資訊。
2. **譯名：現代通行 + 經典混合**（見下方譯名表，**務必照表**）。
3. **全中文化**：對白、敘述、招牌、**介面動詞、物件名**都翻。介面動詞/物件名**用短詞**（顯示空間小）。

## 硬規則（違反會導致回填失敗或遊戲錯亂）

- **每行格式 `TAG<TAB>英文`**。**TAG（如 `[013:LSCR#0200]`）一字不改**，照原樣輸出，只把 TAB 後的英文換成中文。
- **控制碼 `\255\003`、`\255\009\024\000`、`\254\001` 等一律原位保留、數量不變**（它們是換行/等待/插名字/變色）。只翻控制碼「之間」的英文文字。範例：
  `As a matter of fact, I did.\255\003Quite an interesting story` → `說真的，我還真去了。\255\003其實是個滿有意思的故事`
- **`@@@...` padding 原樣保留在尾端**（物件名的填充）：`Cemetery@@@@@` → `墓園@@@@@`。
- **保留字面 ASCII**：字串裡的數字、`ENTER`、`XX`、變數插入符（`\255\007\043\000` 這種）不要動。
- **繁體字要 GBK 打得出來**（GBK 涵蓋絕大多數繁體字；避開極罕見異體字，遇到就換等義常用字）。
- **長度**：對白可自然長短（引擎會自動斷行）；但**介面動詞、物件名、招牌盡量精簡**，別暴增。
- **功能性/資訊性字串**（地名、指令）別硬加笑點；**有發揮空間的對白**才發揮幽默。
- 疑似**開發者名單/致謝**（連續人名、職稱）**保留英文**。

- 輸出**與輸入同行數、同順序**，每行 `TAG<TAB>中文`（TAG 原樣、控制碼/padding 保留）。
